# ❤️ Heart Failure Prediction using Decision Tree

This project uses a Decision Tree machine learning algorithm to predict whether a patient is likely to have heart disease based on medical attributes.

---

## 🚀 Features

* Predicts heart disease (0 = No, 1 = Yes)
* Uses Decision Tree Classifier
* Clean dataset with no missing values
* Easy-to-understand and interpretable model
* Data preprocessing and training included

---

## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:**

  * pandas
  * numpy
  * scikit-learn
  * matplotlib
  * seaborn

---

## 📊 Dataset

The dataset contains the following features:

* Age
* Sex
* ChestPainType
* RestingBP
* Cholesterol
* FastingBS
* RestingECG
* MaxHR
* ExerciseAngina
* Oldpeak
* ST_Slope
* HeartDisease (Target Variable)

✅ No missing values in the dataset.
